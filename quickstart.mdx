## 1. Get your API key

<Steps>
  <Step title="Go to Org Settings">
    Navigate to [app.oneperfectslice.ai/settings](https://app.oneperfectslice.ai/settings) and open the **API Keys** section.
  </Step>
  <Step title="Create a key">
    Click **Create API Key**, give it a name, and select the team it should be scoped to. Your key will be shown once — copy it and store it securely.
  </Step>
</Steps>

<Warning>
  API keys grant full read access to your team's data. Never expose them in client-side code, public repos, or logs.
</Warning>

Your key looks like this: `sk_...` (prefixed with `sk_`).

## 2. List available slices

<CodeGroup>
```bash cURL
curl https://app.oneperfectslice.ai/api/public/v1/slices \
  -H "Authorization: Bearer sk_your_api_key"
```

```python Python
import requests

resp = requests.get(
    "https://app.oneperfectslice.ai/api/public/v1/slices",
    headers={"Authorization": "Bearer sk_your_api_key"}
)
slices = resp.json()["data"]["slices"]
for s in slices:
    print(f"{s['key']}: {s['title']}")
```

```typescript TypeScript
const resp = await fetch(
  "https://app.oneperfectslice.ai/api/public/v1/slices",
  { headers: { Authorization: "Bearer sk_your_api_key" } }
);
const { data } = await resp.json();
console.log(data.slices);
```
</CodeGroup>

```json Response
{
  "data": {
    "slices": [
      {
        "key": "high_intent_deals",
        "title": "High-Intent Deals",
        "objective": "pipeline",
        "description": "Surfaces deals showing strong buying signals.",
        "elementKeys": ["intent_signal", "budget_confirmation"]
      }
    ]
  }
}
```

## 3. Run a slice

Start an async analysis on recent conversations:

<CodeGroup>
```bash cURL
curl -X POST https://app.oneperfectslice.ai/api/public/v1/slices/high_intent_deals/run \
  -H "Authorization: Bearer sk_your_api_key" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: my-unique-key-123" \
  -d '{
    "dateFrom": "2026-03-26T00:00:00Z",
    "dateTo": "2026-04-26T00:00:00Z"
  }'
```

```python Python
import requests

resp = requests.post(
    "https://app.oneperfectslice.ai/api/public/v1/slices/high_intent_deals/run",
    headers={
        "Authorization": "Bearer sk_your_api_key",
        "Content-Type": "application/json",
        "Idempotency-Key": "my-unique-key-123",
    },
    json={
        "dateFrom": "2026-03-26T00:00:00Z",
        "dateTo": "2026-04-26T00:00:00Z",
    },
)
run = resp.json()["data"]
print(f"Run {run['runId']} started — status: {run['status']}")
```

```typescript TypeScript
const resp = await fetch(
  "https://app.oneperfectslice.ai/api/public/v1/slices/high_intent_deals/run",
  {
    method: "POST",
    headers: {
      Authorization: "Bearer sk_your_api_key",
      "Content-Type": "application/json",
      "Idempotency-Key": "my-unique-key-123",
    },
    body: JSON.stringify({
      dateFrom: "2026-03-26T00:00:00Z",
      dateTo: "2026-04-26T00:00:00Z",
    }),
  }
);
const { data } = await resp.json();
console.log(`Run ${data.runId} started — status: ${data.status}`);
```
</CodeGroup>

This returns `202 Accepted` with a `runId`:

```json Response
{
  "data": {
    "runId": 12345,
    "sliceKey": "high_intent_deals",
    "sliceTitle": "High-Intent Deals",
    "status": "running"
  }
}
```

## 4. Poll for results

Check the run status until it completes. Use `suggestedPollAfterSeconds` to determine wait time.

<CodeGroup>
```bash cURL
curl https://app.oneperfectslice.ai/api/public/v1/slice-runs/12345 \
  -H "Authorization: Bearer sk_your_api_key"
```

```python Python
import time, requests

headers = {"Authorization": "Bearer sk_your_api_key"}

while True:
    resp = requests.get(
        "https://app.oneperfectslice.ai/api/public/v1/slice-runs/12345",
        headers=headers,
    )
    run = resp.json()["data"]

    if run["status"] == "completed":
        print("Done!", run["result"])
        break
    elif run["status"] == "failed":
        print("Failed:", run["error"])
        break

    wait = run.get("suggestedPollAfterSeconds", 15)
    print(f"Status: {run['status']} — polling again in {wait}s")
    time.sleep(wait)
```

```typescript TypeScript
const headers = { Authorization: "Bearer sk_your_api_key" };

while (true) {
  const resp = await fetch(
    "https://app.oneperfectslice.ai/api/public/v1/slice-runs/12345",
    { headers }
  );
  const { data: run } = await resp.json();

  if (run.status === "completed") {
    console.log("Done!", run.result);
    break;
  } else if (run.status === "failed") {
    console.log("Failed:", run.error);
    break;
  }

  const wait = (run.suggestedPollAfterSeconds ?? 15) * 1000;
  console.log(`Status: ${run.status} — polling in ${wait / 1000}s`);
  await new Promise((r) => setTimeout(r, wait));
}
```
</CodeGroup>

## 5. Get evidence

Once a run completes, retrieve the supporting evidence:

<CodeGroup>
```bash cURL
curl https://app.oneperfectslice.ai/api/public/v1/slice-runs/12345/evidence \
  -H "Authorization: Bearer sk_your_api_key"
```

```python Python
resp = requests.get(
    "https://app.oneperfectslice.ai/api/public/v1/slice-runs/12345/evidence",
    headers={"Authorization": "Bearer sk_your_api_key"},
)
evidence = resp.json()["data"]
for key, element in evidence["elements"].items():
    print(f"{key}: {element['count']} items")
    for item in element["items"]:
        print(f"  \"{item['quote']}\" (confidence: {item['confidence']})")
```

```typescript TypeScript
const resp = await fetch(
  "https://app.oneperfectslice.ai/api/public/v1/slice-runs/12345/evidence",
  { headers: { Authorization: "Bearer sk_your_api_key" } }
);
const { data } = await resp.json();
for (const [key, element] of Object.entries(data.elements)) {
  console.log(`${key}: ${element.count} items`);
  element.items.forEach((item) =>
    console.log(`  "${item.quote}" (confidence: ${item.confidence})`)
  );
}
```
</CodeGroup>
