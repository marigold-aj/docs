## 1. Get your API key

<Steps>
  <Step title="Go to Org Settings">
    Navigate to [app.oneperfectslice.ai/settings](https://app.oneperfectslice.ai/settings) and open the **API Keys** section.
  </Step>
  <Step title="Create a key">
    Click **Create API Key**, give it a name, and select the team it should be scoped to. Your key will be shown once — copy it and store it securely.
  </Step>
</Steps>

<Warning>
  API keys grant full read access to your team's data. Never expose them in client-side code, public repos, or logs.
</Warning>

Your key looks like this: `sk_...` (prefixed with `sk_`).

## 2. Make your first request

List the slices available to your team:

```bash
curl https://app.oneperfectslice.ai/api/public/v1/slices \
  -H "Authorization: Bearer sk_your_api_key"
```

You should get back a JSON response:

```json
{
  "data": {
    "slices": [
      {
        "key": "high_intent_deals",
        "title": "High-Intent Deals",
        "objective": "pipeline",
        "description": "Surfaces deals showing strong buying signals.",
        "elementKeys": ["intent_signal", "budget_confirmation"]
      }
    ]
  }
}
```

## 3. Run a slice

Start an analysis on recent conversations:

```bash
curl -X POST https://app.oneperfectslice.ai/api/public/v1/slices/high_intent_deals/run \
  -H "Authorization: Bearer sk_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "dateFrom": "2026-03-26T00:00:00Z",
    "dateTo": "2026-04-26T00:00:00Z"
  }'
```

This returns `202 Accepted` with a `runId`:

```json
{
  "data": {
    "runId": 12345,
    "sliceKey": "high_intent_deals",
    "sliceTitle": "High-Intent Deals",
    "status": "running"
  }
}
```

## 4. Poll for results

Check the run status until it completes:

```bash
curl https://app.oneperfectslice.ai/api/public/v1/slice-runs/12345 \
  -H "Authorization: Bearer sk_your_api_key"
```

When `status` is `"completed"`, the `result` field contains your structured insights. Use the `suggestedPollAfterSeconds` field to determine how long to wait between polls.

## 5. Get evidence

Once a run completes, retrieve the supporting evidence:

```bash
curl https://app.oneperfectslice.ai/api/public/v1/slice-runs/12345/evidence \
  -H "Authorization: Bearer sk_your_api_key"
```
